# 📄 ChatGPT-Inspired Documents Storing & Retrieval System (DocuGuardian)

A smart document management system inspired by ChatGPT, where users can securely store and retrieve their personal documents using a conversational interface.

🔗 **Live Demo:** https://chat-gpt-documents-storing.vercel.app/
🔗 **GitHub Repo:** https://github.com/yahskamrahs/ChatGPT-Inspired-Documents-Storing

---

## 🚀 Features

* 🔐 User Authentication (Login System)
* 🤖 ChatGPT-like Interface for interaction
* 📂 Store important documents securely
* 🔍 Retrieve documents using simple chat commands
* ⚡ Fast and responsive UI
* 🌐 Deployed on Vercel

---

## 🧠 How It Works

1. User logs into the system using credentials
2. A chatbot interface (DocuGuard) greets the user
3. User requests documents like:

   * Aadhaar Card
   * PAN Card
   * Passport Photo
   * Electricity Bill
   * Income Certificate
4. The system fetches and returns the requested document instantly
